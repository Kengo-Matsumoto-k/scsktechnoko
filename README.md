### github の使い方
1.branch を切る
`git branch ~~`
`git checkout ~~`
2.編集をする
3.`git add .`
4.`git commit -m "commit名"`
5.`git push -u origin branch名`
6.Pull Request を出す
7.conflict がなければ merge
※conflict が出たとき、修正できるのであれば、修正後に merge
8.merge したら、毎回`git pull origin main`で自分の環境を最新版にしてください
